| Version | Date | Notes |

### Conventions

- **Filename format:** `<document>-v<major>.<minor>.pdf`
- **Major bump:** Structural changes, new sections, removed content
- **Minor bump:** Corrections, clarifications, formatting
- When releasing a new version, keep the previous version in `docs/archive/` for reference
